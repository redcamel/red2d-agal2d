public function loadFile($urlPath:String, $completeMethod:Function = null, $useTrace:Boolean = true):*

파일을 로드하는 매서드

Parameters

$urlPath:String — 로드 할 파일의 경로
 
$completeMethod:Function (default = null) — 로딩완료시 체크할 메서드
 
$useTrace:Boolean (default = true) — 로드 결과를 trace 할 것 인지를 설정 / 최종 산출물에서는 false로 설정하는 것이 좋다. 

			 IMGLoader Event : open 발생 || 대상 : ./res/testImg.jpg
			 IMGLoader Event : complete 발생 || 대상 : ./res/testImg.jpg
위와같은 trace값의 출력여부를 설정한다.
